## Run Locally

1. Clone the repository

```bash
  git clone https://link-to-project
```

2. Install dependencies

```bash
  npm install
```

3. Paste in token (index.js | line 6)

```bash
  token: "YOUR_TOKEN",
```

4. Change things in help.js (optional)
```bash
  "...le[(BOT_NAME)'s com..."
  "...tion[(BOT_PREFIX)achieve - Gen..."
```

5. Start the bot

```bash
  node index.js
```
